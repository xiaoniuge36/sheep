# 🐑 羊了个羊 (Web版)

这是一个使用 HTML/CSS/JS 开发的“羊了个羊”风格的 Web 小游戏，包含关卡难度递增、3D 卡片效果、排行榜（模拟）等功能。

## 🚀 如何在中国大陆免费部署

由于 Surge 等国外静态托管服务（如 GitHub Pages, Netlify, Vercel）在中国大陆访问速度不稳定或被墙，推荐使用 **Gitee Pages (码云)** 进行部署。它是国内服务，访问速度极快且免费。

### 步骤 1：注册 Gitee 账号
访问 [https://gitee.com/](https://gitee.com/) 并注册一个账号。

### 步骤 2：创建仓库
1. 登录后，点击右上角的 `+` 号，选择 **“新建仓库”**。
2. 仓库名称填入 `sheep-game`（或任意名称）。
3. 选择 **“开源”**（Gitee Pages 必须是开源仓库才能免费使用）。
4. 点击 **“创建”**。

### 步骤 3：上传代码
将本项目文件夹中的所有文件（`index.html`, `style.css`, `script.js`, `README.md`）上传到仓库。

**如果你不会使用 Git 命令，可以直接在网页端上传：**
1. 进入刚创建的仓库页面。
2. 点击 **“文件”** -> **“上传文件”**。
3. 将本地文件夹中的文件拖拽进去并提交。

### 步骤 4：开启 Pages 服务
1. 在仓库页面，点击顶部菜单栏的 **“服务”** -> **“Gitee Pages”**。
2. **实名认证**：如果是首次使用，根据国家法规，需要上传身份证进行实名认证（通常 1 个工作日审核）。
3. 审核通过后，选择部署分支（通常是 `master` 或 `main`），点击 **“启动”**。
4. 启动成功后，它会给出一个网址（例如 `https://yourname.gitee.io/sheep-game`）。

### 5. 分享给朋友
直接复制那个网址发给微信好友即可畅玩！

## 💾 关于数据存储

目前的“排行榜”数据是存储在玩家浏览器本地（LocalStorage）的，仅供演示。
如果需要实现**全网真实联机排行榜**，您需要一个后端数据库。

推荐方案（免费）：
- **LeanCloud** (国内版)：提供免费的 Serverless 数据库，适合前端直接调用。
- **微信云开发**：如果是开发微信小程序，这是最佳选择。
